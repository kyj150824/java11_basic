1. zulu jdk11 다운로드
2. (java version operating sytem architercuture 등 고려) - zulu11 ~ win_x64.msi로 다운로드 후 설치
3. 개발 실행 환경을 위하여 윈도우 탐색기를 실행하고, 내 pc 에서 마우스 오른쪽 버튼을 누르고 나오는 메뉴에서 속성을 선택
4. 고급시스템 설정 - 고급탭의 환경변수 선택 - 시스템 변수 항목의 새로 만들기 선택 - 변수이름 JAVA_HOME -변수 값 디렉터리찾아보기 - C드라이브 - 프로그램 파일 - Zulu - zulu11 선택
5. 시스템변수 항목에서 Path 항목 편집 선택 후 zulu-11 클릭 후 %JAVA_HOME%\bin\로 수정하고, 차례대로 모든 창의 확인 버튼을 눌러 개발환경 설정.
